# aiostack_devenv Cookbook

a Chef cookbook to install the staging environement for AIOstack

## Requirements


### Platforms

- Ubuntu

### Chef

- Chef 12.0 or Later

### Cookbooks

- `poise-python` - aiostack_devenv needs  poise-python.
